The Controls are as follows:
Slider - Changes the parametric values of the seashell model
Arrow keys - Rotates the model
Plus and Minus keys - Zoom in and out

Wireframe render - Renders the model in wireframe
Phong render - Renders the model in solid 3d with Phong shading
Gouraud render - Renders the model in solid 3d with Phong shading
Texture render - Non functional
